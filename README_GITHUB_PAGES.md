## GitHub Pages
1. Pakk ut zip-filen.
2. Opprett et nytt public repository på GitHub, f.eks. `whiskylogg`.
3. Last opp alle filene i mappen til repoets rot.
4. Gå til **Settings → Pages**.
5. Velg **Deploy from a branch**.
6. Velg `main` og `/ (root)`.
7. Vent 1–3 minutter.
8. Åpne GitHub Pages-lenken i Safari.
9. Trykk **Del → Legg til på Hjem-skjerm**.

## Viktig
Data lagres lokalt på enheten. Ta backup fra appens Backup-side med jevne mellomrom.
