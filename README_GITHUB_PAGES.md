# Whiskylogg v1.8

Denne versjonen inneholder:
- Bedre startside med:
  - beste kjøp
  - bør veies nå
  - lav beholdning
  - sist smakt
  - volum per type
- Bilder på type- og flaskekort
- Kun full vekt som nødvendig input i bibliotek
- Tom vekt beregnes automatisk
- Tetthet beregnes tabellbasert med lineær interpolasjon mellom tabellpunkter ved 20 °C
- Smakinger med snittscore per flaske
- Backup/import

Tetthet:
Det brukes en tabell for 0–60 % ABV ved 20 °C, og det interpoleres lineært mellom punktene.

Oppdatering:
1. Pakk ut zip-filen
2. Erstatt filene i GitHub-repoet ditt
3. Commit
4. Vent 1–2 minutter og åpne appen igjen
