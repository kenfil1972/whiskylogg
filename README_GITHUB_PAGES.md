# Whiskylogg v1.6 full

Innhold:
- Bedre startside med kort for total kost, total volum, beste kjøp, lav beholdning og sist smakt
- Bibliotek med TBA/tomt felt for tom vekt
- Automatisk beregning av tom vekt fra full vekt - volum × egenvekt
- Flasker med batch og flaske nr
- Veiing per flaske
- Smakinger med snittscore per flaske
- Backup/import

Oppdatering:
1. Pakk ut zip-filen.
2. Erstatt alle filene i GitHub-repoet ditt med disse.
3. Commit.
4. Vent 1-2 minutter og åpne appen på nytt.
5. Hvis gammel versjon henger igjen på iPhone, oppdater siden eller fjern hjemskjerm-appen og legg den til på nytt.
